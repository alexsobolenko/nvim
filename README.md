# Installation

1. Install `composer`
2. Install `nodejs`
3. Install following npm packages globally:
    * `tree-sitter-cli`
    * `intelephense`
    * `@volar/server`
    * `vscode-langservers-extracted`
    * `dockerfile-language-server-nodejs`
    * `typescript-language-server`
    * `typescript`
    * `yaml-language-server`
    * `jsregexp`
4. Install `lazygit` to use `lazygit` plugin
